# yyaml

![yyaml-logo](docs/images/yyaml.svg)
yyaml is currently the fastest implementation of this library and the available codebase is written in C. The roadmap includes ports for C++, Python, and Go, but today only the C version is published.

- Current implementation: C
- Planned languages: C++, Python, Go
- Inspired by the source code of [yyjson](https://github.com/ibireme/yyjson)

Repository: [github.com/mohammadraziei/yyaml](https://github.com/mohammadraziei/yyaml)
